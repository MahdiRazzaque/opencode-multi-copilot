## Reading order

For a first pass through the project, read the documents in numeric order. For task-specific work:

- Start with `02-installation-and-setup.md` for local setup
- Start with `04-configuration-reference.md` when editing user-facing config
- Start with `06-interface-reference.md` when changing public behaviour or integration points
- Start with `07-development-and-testing.md` when contributing code
